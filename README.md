# CodeAlpha_Tasks
## CodeAlpha Python Internship Projects
Hello! The projects I created while working as a Python programming intern at CodeAlpha are included in this repository.

I concentrated on honing my Python fundamentals, problem-solving techniques, and comprehension of how straightforward programs can address practical issues during this internship.
## Projects I Worked On
## Chatbot
This is a simple rule-based chatbot that can interact with users through predefined responses. It reacts to simple inputs, such as salutations and frequently asked queries.

What I learned: I knew how to mimic talks using loops, functions, and conditional statements (if-elif).

## Hangman Game
A fun text-based game where the user guesses a hidden word one letter at a time. The player has limited chances to guess the word correctly.

What I learned: I practiced working with strings, lists, loops, and the random module while building a small game.

## Stock Portfolio Tracker
This project helps calculate the total value of stocks based on user input and predefined prices.

What I learned: I gained knowledge about how to use dictionaries, accept user input, and carry out simple computations. It also helped me understand how basic programs may be used to activities relating to finance.
