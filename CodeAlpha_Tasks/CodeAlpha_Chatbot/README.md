## Chatbot

This project is a simple rule-based chatbot built using Python. It communicates with users by providing predefined responses, making it a good introduction to basic conversational logic.

### Features

* Replies to greetings such as "hello" and "hi"
* Handles basic user queries
* Delivers preset responses
* Offers an interactive command-line experience

### Concepts Used

* `if-elif` conditional statements
* `while` loops for continuous interaction
* Functions for better code organization
* Basic string handling

### How to Run

* Ensure Python is installed on your system
* Execute the following command in your terminal:
  `python chatbot.py`

### About

This project was developed as part of the CodeAlpha Python Internship, with a focus on core programming concepts and beginner-level chatbot development.
