## Hangman Game

A basic command-line Hangman game where players try to guess a hidden word one letter at a time.

## Features

* Randomly selects a word for each game
* Provides a fixed number of attempts
* Shows the current progress after every guess

## Concepts Used

* `random` module
* Loops
* String and list operations

## How to Run

python hangman.py
