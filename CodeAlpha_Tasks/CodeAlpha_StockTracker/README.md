## Stock Portfolio Tracker

This project focuses on calculating the total investment value using user input along with predefined stock prices.

## Features

* Enables users to input stock names and their respective quantities
* Computes the overall value of the portfolio
* Simple, clear, and user-friendly

## Concepts Used

* Dictionaries
* Input and output handling
* Basic arithmetic operations

## How to Run

python stock_tracker.py
